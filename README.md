# bugs-watcher 

WS-server for bugs workflow watching for Bugzilla 5.0+


Installing:

Clone the repo:  

```bash
git clone https://gitlab.eterfund.ru/kantegory/bugs-watcher.git
```

Move to directory:
```bash
cd bugs-watcher/server
```

Install all deps:

```bash
npm install
```

Start server:

```bash
npm start
```

Server is available now at http://hostname:17003
